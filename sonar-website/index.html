Installation

System Requirements

  • Java 11 or higher
  • Spigot 1.16.5 or newer
  • At least 512MB free RAM

Installation Steps

Step 1: Download the latest Sonar.jar file

Step 2: Place the JAR file in your server's plugins folder

plugins/ ‚îî‚îÄ‚îÄ Sonar.jar

Step 3: Restart your server

stop # Wait a few seconds # Start your server again

Step 4: Configure the plugin in plugins/Sonar/config.yml

Basic Configuration

sonar: # Enable/disable the plugin enabled: true # Server name server-name: "MyServer" # Enable detailed logging debug: false # Features to enable features: monitoring: true player-tracking: true auto-save: true

Documentation

Commands

/sonar status - View current server status and statistics

/sonar reload - Reload the plugin configuration

/sonar info [player] - Get detailed information about a player

/sonar debug - Toggle debug mode (admin only)

Permissions

sonar.admin - Access to all admin commands

sonar.status - View server status

sonar.reload - Reload configuration

sonar.info - View player information

Troubleshooting

Plugin not loading?

  • Check if Java version is 11 or higher
  • Check server console for error messages
  • Verify the JAR file is in the correct plugins folder

High memory usage?

  • Disable unnecessary features in config.yml
  • Reduce logging level from DEBUG to INFO
  • Check for plugin conflicts with other plugins
